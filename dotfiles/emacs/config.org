* Introduction

* Package Management
I am using the built-in package manager and =use-package= to manage my extensions.

#+begin_src emacs-lisp
  (require 'package)
  (add-to-list 'package-archives '("melpa" . "https://melpa.org/packages/") t)
  (package-initialize)
  
  (require 'use-package)
  (setq use-package-always-ensure t
        use-package-always-defer t)
#+end_src

* Core System Settings
Here I ensure my Fish shell paths are pulled into Emacs correctly so external tools work seamlessly within Wayland.

#+begin_src emacs-lisp
  (use-package exec-path-from-shell
    :demand t
    :config
    (setq exec-path-from-shell-shell-name "/usr/bin/fish")
    (setq exec-path-from-shell-variables '("PATH" "MANPATH"))
    (exec-path-from-shell-initialize))
#+end_src
